# Trees

Types of trees :

- Binary Trees
- AVL Trees

Types of traversals for trees :

- Preorder traversal -> Node, Left, Right
- Postorder traversal -> Left, Right, Node
- Inorder traversal -> Left, Node, Right [**Inorder traversal exists only for binary trees**]
